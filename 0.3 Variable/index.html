Variables in JS

A javaScript variable is just a name of storage location

And we can give thatr name in thre  ways.
  • Using let keyword.
  • Using const keyword.
  • Using var keyword.
Example:
  let firstName = 'Aniket'
  let age = 21
  let isHappy = true

  let userIntro = 'Hi! my  name is ' + firstName + 'and  I am ' + age +'years old.'

  firstName = 'Chauhan'

const value can't be change further and initialization is compulsary.

let and const is block scope variable.

var is global scoped variable.

let and const must be initialized before use, otherwise it well give error but in case if var it will give undefined(it's a value).

rules for naming variables

A variable contains any for the following characters but must not start with a digit

  1. Capital letters: A to Z
  2. small letters: a to z
  3. Digits: 0 to 9(but we can't starts with it.)
  4. only the special character used: $ and _
