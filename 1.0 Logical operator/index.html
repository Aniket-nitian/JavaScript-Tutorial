Logical Operators

  • && (And)
  • || (Or)
  • ! (Not)
      0 && 2 -->0  (AND operator give FALSY when ever it get. in all other case it will give the second expression result.)  
    
      0 || 2 -->2    (OR Operator give value when ever it get a TRUTHY value, not try to check other expression.)

      'Hello' && console.log('hello') -->undefined (console.log('hello1') value is undefined)

      boolean(0 || 2) -->true
