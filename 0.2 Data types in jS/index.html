Data types in JAVASCRIPT

To check data type --> typeof dataType

There are two type of Data types in JS.

  1. Primitive.
  2. Non primitive.
  We can classify primitive data types in 7 categories
  1. number.
  2. string.
  3. boolean(true,false).
  4. undefined.
  5. null(object).
  6. bigint.
  7. symbol. 
  parseInt('string') is used to convert string into number.
  any number + '' is used to conver number into a string.
